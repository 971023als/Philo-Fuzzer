# Philo-Fuzzer 🏛️

Philo-Fuzzer는 다양한 철학적 관점(렌즈)을 사용하여 생성형 AI 모델을 평가, 테스트 및 강화하도록 설계된 **운영 환경용 AI 윤리 레드팀 파이프라인(Harness)**입니다. 니체, 하이데거, 카뮈, 소크라테스 등 역사상 가장 위대한 사상가들을 모방한 멀티 에이전트 시뮬레이션을 사용하여 AI 시스템의 실존적 위험, 윤리적 취약점, 논리적 오류를 포착합니다. 단순한 안전성 검사를 넘어 심층적인 윤리 규정 준수 감사를 수행합니다.

## 주요 기능 🚀
- **13개의 철학자 에이전트 렌즈**: 자율성, 비인간화, 실존적 피해, 논리 등 다양한 윤리적/철학적 프레임워크를 기반으로 AI 출력을 분석합니다.
- **자동 가드레일 및 증거 계층화**: 근거 없거나 환각(Hallucination)에 의한 AI 발견 사항을 탐지하고 등급을 하향 조정하여 오탐지를 방지합니다.
- **강력한 중재자(Arbiter) 충돌 해결**: 상이한 철학적 프레임워크 간의 상충되는 해석을 지능적으로 처리합니다.
- **컴플라이언스 및 감사 대응**: 위험 컨텍스트 및 정책 참조와 매핑된 표준화 및 추적 가능한 스키마 출력(JSON/MD)을 생성합니다.

## 시작하기 ⚙️
Python 3.10 이상이 설치되어 있어야 합니다.

```bash
# 레포지토리 클론
git clone https://github.com/971023als/Philo-Fuzzer.git
cd Philo-Fuzzer

# 메인 시스템 실행 (Mock 테스트)
python ethical_redteam_harness/main.py
```
