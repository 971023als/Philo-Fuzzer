# Philo-Fuzzer 🏛️

Philo-Fuzzerは、様々な哲学的視点（レンズ）を使用して生成AIモデルを評価、テスト、および強化するために設計された**実運用対応のAI倫理レッドチームハーネス**です。歴史上の偉大な思想家（ニーチェ、ハイデガー、カミュ、ソクラテスなど）を模倣したマルチエージェント・シミュレーションを採用することで、AIシステムにおける実存的リスク、倫理的脆弱性、論理的誤謬を浮き彫りにし、単なる安全性チェックを超えた深い倫理的コンプライアンス監査を実現します。

## 主な機能 🚀
- **13の哲学者エージェント・レンズ**: 自律性、非人間化、実存的危害、論理など、様々な倫理的・哲学的枠組みからAIの出力を分析します。
- **自動化されたガードレールと証拠の階層化**: 根拠のないAIの発見やハルシネーション（幻覚）を検出し、その評価を下げることで、誤検知を防止します。
- **堅牢な調停者（Arbiter）による競合解決**: 異なる哲学の枠組み間で生じる相反する解釈をインテリジェントに処理します。
- **コンプライアンスと監査への対応**: リスクコンテキストとポリシー参照にマッピングされた、標準化され追跡可能なスキーマ出力（JSON / Markdown）を生成します。

## はじめに ⚙️
Python 3.10以上がインストールされていることを確認してください。

```bash
# リポジトリのクローン
git clone https://github.com/971023als/Philo-Fuzzer.git
cd Philo-Fuzzer

# エンジンの実行 (Mockテスト)
python ethical_redteam_harness/main.py
```

## サポートされているエージェント
- **ニーチェ (Nietzsche)**: 自律性、権力力学、自己欺瞞
- **ハイデガー (Heidegger)**: 非人間化、道具化、非本来性
- **カミュ / サルトル (Camus / Sartre)**: 実存的危害、不条理
- **ソクラテス (Socrates)**: 論理的一貫性と前提の根拠
- *(その他のエージェントも開発中...)*
